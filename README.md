# TP-Parte-I---Rhenner-Santos-e-Lucas-Stefanelli

Este é um sistema produzido para a disciplina de Estruturas de Dados, ministrada pelo docente Thiago Paixão. O objetivo desse programa é organizar os dados de um campeonato de futebol, permitindo que o usuário possa ler e modificar informações do sistema (função que será implementada na segunda parte do trabalho). Para isso, um menu será mostrado no terminal e algumas opções aparecerão em um formato de menu, para facilitar a interação do usuário.
