# TP-Parte-I---Rhenner-Santos-e-Lucas-Stefanelli

Este é um sistema produzido para a disciplina de Estruturas de Dados, ministrada pelo docente Thiago Paixão. O objetivo desse programa é organizar os dados de um campeonato de futebol, permitindo que seja possível consultar e modificar informações do banco de dados (a segunda função será implementada na parte 2 do trabalho). Para isso, um menu será mostrado no terminal com algumas opções, para que o usuário possa interagir com o sistema.
